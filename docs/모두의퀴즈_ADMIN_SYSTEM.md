# 모두의퀴즈 — 어드민 시스템 설계 (ADMIN_SYSTEM.md)

---

## 1. 어드민 기능 전체 목록

### 1.1 접근 제어
- Firebase Auth + Firestore `users.role === 'admin'` 검증
- 미들웨어에서 `/admin/*` 경로 보호
- 어드민 API는 추가로 Service Account 또는 `ADMIN_SECRET` 헤더 검증

### 1.2 기능 매트릭스
| 기능 | 자동 | 수동 | 설명 |
|------|------|------|------|
| RSS 뉴스 수집 | ✅ | ✅ | Cron 또는 버튼 클릭 |
| 기사 요약 생성 | ✅ | ✅ | AI 자동 또는 수동 입력 |
| AI 퀴즈 생성 | ✅ | ✅ | 선택 기사 기반 |
| 퀴즈 수동 작성 | - | ✅ | 객관식/주관식/이미지/영상 |
| 퀴즈 배포 스케줄 | ✅ | ✅ | 시간 지정 or 즉시 |
| 사용자 관리 | - | ✅ | 조회/제재/등급 |
| 랭킹 통계 | ✅ | - | 자동 집계 대시보드 |
| 공지/이벤트 | - | ✅ | 팝업/배너 |

## 2. 어드민 대시보드 상세

### 2.1 메인 대시보드 (`/admin`)

#### 실시간 KPI 카드
```typescript
// 표시 지표
{
  todayCollected: number;       // 오늘 수집된 기사 수
  todayQuizGenerated: number;   // 오늘 생성된 퀴즈 수
  todayPlayers: number;         // 오늘 퀴즈 참여자 수
  totalUsers: number;           // 전체 가입자 수
  newUsersToday: number;        // 오늘 신규 가입자
  publishedSets: number;        // 배포된 퀴즈셋 수
  pendingSets: number;          // 배포 대기 퀴즈셋 수
  avgScore: number;             // 오늘 평균 점수
}
```

#### 수집 현황 테이블
| 카테고리 | 수집 기사 | AI 요약 | 퀴즈 생성 | 상태 |
|---------|---------|---------|---------|------|
| 연예 | 28 | ✅ | ✅ 100문제 | 배포완료 |
| K팝 | 25 | ✅ | ✅ 100문제 | 배포완료 |
| AI | 22 | ✅ | ⏳ 생성중 | 진행중 |
| 정치 | 30 | ✅ | ❌ 실패 | 오류 |

#### 빠른 액션 버튼
- `[🔄 RSS 수동 수집]` — 선택 카테고리 즉시 수집
- `[🤖 AI 퀴즈 생성]` — 선택 카테고리 퀴즈 생성
- `[📢 즉시 배포]` — draft 퀴즈셋 즉시 published
- `[📊 오늘 랭킹 확인]` — 랭킹 현황 팝업

### 4.3 수동 문제 작성 (`/admin/quiz/create`)

#### 4.3.1 문제 유형별 에디터

**객관식 4지선다**
```
┌─────────────────────────────────┐
│  유형: [객관식 ▼]               │
│                                 │
│  카테고리: [K팝 ▼]             │
│  연령대:   [20대 ▼]            │
│  난이도:   ○쉬움 ●보통 ○어려움  │
│                                 │
│  문제:                          │
│  ┌───────────────────────────┐ │
│  │ 2024년 그래미 어워드에서  │ │
│  │ K팝 최초로 수상한 팀은?   │ │
│  └───────────────────────────┘ │
│                                 │
│  보기:                          │
│  ① [BTS                      ] │
│  ② [블랙핑크                  ] │
│  ③ [뉴진스                   ] │
│  ④ [aespa                    ] │
│                                 │
│  정답: [① ▼]                  │
│  해설: [BTS는 ~했기 때문에...  ] │
│                                 │
│  관련 뉴스: [검색 또는 직접입력] │
└─────────────────────────────────┘
```

**OX 퀴즈**
```
│  유형: [OX ▼]                  │
│  문제: [...........................] │
│  정답: ○O  ○X                  │
│  해설: [...........................] │
```

**단답형 주관식**
```
│  유형: [단답형 ▼]              │
│  문제: [...........................] │
│  정답: [_____________]         │
│  허용 유사 정답:               │
│  [+ 추가]                      │
│  해설: [...........................] │
```

**이미지형**
```
│  유형: [이미지 ▼]              │
│  이미지: [파일 업로드 / URL]   │
│          [미리보기]            │
│  문제: [...........................] │
│  보기 & 정답: (객관식과 동일)  │
```

**영상 연동형**
```
│  유형: [영상 ▼]               │
│  YouTube URL: [____________]  │
│  시작(초): [___] 끝(초): [___]│
│  문제: [...........................] │
│  보기 & 정답: (객관식과 동일)  │
```

## 6. 통계 대시보드 (`/admin/ranking`)

### 6.1 참여 통계
```
[일간/주간/월간] 차트:
  - 일별 퀴즈 참여자 수 (라인 차트)
  - 카테고리별 참여율 (파이 차트)
  - 연령대별 분포 (바 차트)
  - 지역별 분포 (지도 or 바 차트)
```

### 6.2 퀴즈 성과
```
- 카테고리별 평균 정답률
- 문제 유형별 정답률
- 가장 많이 틀린 문제 TOP 10
- 평균 풀이 시간
```
