# 모두의퀴즈 — 비즈니스 모델 & 성장 전략 (BUSINESS.md)

---

## 1. 서비스 포지셔닝

### 1.1 타겟 시장
- **1차 타겟**: 뉴스에 관심 있는 20-40대 한국인
- **2차 타겟**: K팝/AI/IT에 관심 있는 10-20대
- **3차 타겟**: 지식 경쟁을 즐기는 모든 연령대

### 1.2 차별화 포인트
| 경쟁 서비스 | 모두의퀴즈 차이점 |
|-----------|----------------|
| 네이버 뉴스 | 단순 소비 → 능동적 퀴즈 게임화 |
| 카카오 브레인 | 퀴즈 중심 + 뉴스 기반 실시간성 |
| 일반 퀴즈앱 | 매일 새 뉴스 기반 + 시의성 있는 콘텐츠 |
| 유튜브 시사채널 | 양방향 참여 + 랭킹 경쟁 |

## 2. 수익화 모델 (단계별)

### Phase 1 (0-6개월): 무료 서비스, 유저 확보
- 모든 기능 무료
- 목표: 월간 활성 사용자 10,000명

### Phase 2 (6-12개월): 스폰서십 도입
```
스폰서 배너:
  - 퀴즈 완료 화면에 스폰서 브랜드 노출
  - CPM 기반 수익

스폰서 퀴즈:
  - "OO브랜드 협찬 K팝 퀴즈"
  - 브랜드 인지도 + 재미 결합
  - 패키지 가격: 월 100만원~

대회 타이틀 스폰서:
  - "OO배 모두의퀴즈 왕중왕전"
  - 대회 상금 스폰서 + 브랜드 노출
```

### Phase 3 (12개월+): 프리미엄 구독
```
무료 티어:
  - 데일리 퀴즈 1회
  - 카테고리 퀴즈 2개/일
  - 기본 랭킹

프리미엄 티어 (월 4,900원):
  - 광고 없음
  - 무제한 카테고리 퀴즈
  - 상세 성적 분석
  - 오답 노트
  - 힌트 5회/일
  - 전용 뱃지
```

### Phase 4: 포인트 현금화
```
포인트 → 현금/상품권:
  - 스폰서 경품 응모권
  - 편의점 상품권 교환
  - 제휴 할인 쿠폰
```

## 3. 성장 전략

### 3.1 바이럴 메커니즘
```
핵심 루프:
사용자 A 퀴즈 완료
    → 카카오 공유 (자랑 심리)
    → 사용자 B 링크 클릭
    → 즉시 퀴즈 플레이
    → B도 공유
    → 지수적 확산

강화 요소:
  - "나보다 잘 풀 수 있어?" 도발적 카피
  - 점수 비교 화면 (공유자 vs 나)
  - 카카오 오픈채팅 "퀴즈 대결방" 기능 (향후)
```

### 3.2 커뮤니티 형성
```
지역별 커뮤니티:
  - 제주 VS 서울 랭킹 대결
  - 지역 대표 선발전

연령대별 자존심 대결:
  - "20대가 30대를 이겼다!" 뉴스피드
  - 연령대별 위클리 챔피언 발표

카테고리 마스터:
  - K팝 퀴즈 전국 1위 = "K팝 마스터" 칭호
  - SNS 공유 유도
```

### 3.3 미디어/파트너십 전략
```
뉴스 미디어:
  - 언론사와 제휴 → 독점 퀴즈 콘텐츠
  - "OO일보 × 모두의퀴즈 시사왕"

K팝 레이블:
  - SM, HYBE, YG 아이돌 관련 공식 퀴즈
  - 팬덤 유입 기회

교육 기관:
  - 학교 시사교육 도구
  - 방과후 수업 연계
```

## 5. 대회 이벤트 캘린더

```
매일:    데일리 퀴즈 (전체 랭킹)
매주:    위클리 챌린지 (연속 참여 보너스)
매월:    먼슬리 왕중왕전 (월간 TOP 100 초대)
분기:    카테고리별 챔피언십 (K팝왕, AI왕 등)
연간:    이얼리 챔피언십 "시사 왕중왕" (연간 TOP 50)

특별 이벤트:
  - 수능 시즌: 수능 상식 특집
  - 연말: "올해의 뉴스 퀴즈 대전"
  - 월드컵/올림픽: 스포츠 특집 대회
```
