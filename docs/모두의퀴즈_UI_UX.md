## 3. 모바일 UX 가이드라인

### 3.1 터치 인터랙션
- 퀴즈 보기 선택: 전체 영역 터치 가능 (최소 48px 높이)
- OX 퀴즈: 좌측 절반(X) / 우측 절반(O) 터치
- 스와이프: 결과 화면에서 다음 문제로 스와이프 가능

### 3.2 애니메이션 사양
```
정답 효과:    초록 플래시 + 체크마크 애니메이션 (0.3s)
오답 효과:    빨간 진동 (shake) + X 표시 (0.3s)
콤보 효과:    🔥 이모지 + 점수 팝업 (0.5s, bounce)
타이머 긴급:  10초 이하시 빨간색 + 진동 (CSS animation)
점수 카운트업: 결과 화면에서 숫자 카운팅 애니메이션
```

### 3.3 카카오 공유 딥링크
```
공유 URL: https://moquiz.kr/quiz/[setId]?ref=[userId]

수신자 플로우:
  - 비로그인: 랜딩 페이지 → "구글 로그인 후 퀴즈 시작" → 동일 퀴즈로 redirect
  - 로그인: 즉시 퀴즈 시작 → 완료 후 "공유자 점수: X점 vs 내 점수: Y점" 비교 화면
```

## 4. 반응형 브레이크포인트

```css
/* 모바일 우선 */
default:  320px ~  (기본 모바일)
sm:       640px ~  (태블릿 소형)
md:       768px ~  (태블릿)
lg:       1024px ~ (데스크탑 소형)
xl:       1280px ~ (데스크탑) → 어드민 최적화
```

퀴즈 게임 자체는 모바일 중심 (최대 너비 480px 고정, 가운데 정렬).
어드민 대시보드는 1024px+ 데스크탑 최적화.
